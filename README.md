# 🌱Commit Grow

> GitHub 활동 수집 하고 AI가 도와주는 회고 작성으로 개발자가 성장하는 플랫폼 입니다.

## 1. 프로젝트 소개

혼자 회고 작성하는 것이 타인의 피드백이 없어 지속되기 어렵다는 문제점에서 개발자가 매일의 성장을 기록하고 공유하는 것으로 문제를 해결하자는 생각으로 출발하였습니다.

GitHub 활동을 자동 수집·집계하여 데이터 기반 회고를 돕고, 활동 통계와 랭킹으로 성장을 가시화하는 **개발자 활동 분석 플랫폼**입니다.

단순 기록 게시판이 아니라, "수집 → 집계 → 인사이트"로 이어지는 데이터 흐름을 중심에 둡니다.

## 3. 핵심 기술 챌린지

직접 조사 후 블로그 글을 링크하여 작성
포맷: 문제 → 해결 → 목표/검증

예상 챌린지 후보:

- Git Webhook 멱등성 + 재시도 설계
- LLM 스트리밍 + 회고 배치 처리
- K6 부하 테스트 후 개선사항
- 이메일 발송 실패 복구

작성 시 주의:

- "결과"는 반드시 측정 가능한 수치 또는 검증 방법으로
- 아는 만큼만 작성 (방어 못 할 표현 금지)

## 5. 주요 기능

### GitHub 연동 & 활동 수집

- OAuth 로그인 + 커밋/PR 활동 자동 수집 파이프라인
- 멱등성 보장으로 중복 집계 방지
- 비공개 레포 커밋 수만 집계 (메시지 마스킹)

### AI 회고

- 수집된 당일 활동 자동 집계 → LLM 스트리밍 회고 초안 생성
- 액션 포인트 관리 (이전 회고 연속성)

### 활동 분석

- 주간 활동 랭킹으로 다른 개발자와 성장을 비교
- 잔디 스타일 활동 히트맵으로 기간별 활동 시각화
- 연속 회고 스트릭 배지

### 주간 리포트

- 한 주간 활동 요약 + 회고 모아보기 제공
- 정해진 시간에 자동 발송되는 주간 이메일 리포트

### 커뮤니티

- 공개 회고 피드 조회
- 댓글/좋아요 동기 집계
