# [1단계] Commit Grow 디자인 토큰 확정 프롬프트

Next.js + Tailwind + shadcn/ui 기반 개발자 성장 플랫폼 "Commit Grow"의 디자인 시스템 토큰을 만들어줘. 화면은 아직 만들지 말고, 아래 조건으로 색상/타이포/컴포넌트 스타일 가이드만 먼저 확정해줘.

**플랫폼**: 데스크톱 웹 전용 (반응형 대응 없음), 기준 화면 폭 1440px

**색상**
- 포인트: #18F293
- 배경: #FFFFFF
- 텍스트(검은색): #3E3E40
- 구분 영역(회색 처리): #FBFBFC
- 카드 등(다른 흰색): #FEFEFE

**폰트**: Pretendard

**타입 스케일**
- H1 (페이지 제목): 24px, 700
- H2 (섹션 제목): 18px, 600
- Body (본문): 14px, 400
- Caption (보조 텍스트): 12px, 400

**요청**
- 버튼(primary/secondary), 카드, 뱃지, 모달 기본 스타일 가이드를 위 토큰 기준으로 정의
- radius, spacing 등 기본값도 함께 제안
- 이 스타일 가이드를 이후 모든 화면 생성에 고정 기준으로 사용할 것
